문서내에서 스크립트 선언후 코드를 구현해서 호출하는 방법입니다.

함수(function)를 호출해서 이 내용이 바뀝니다.

이번엔 스크립트를 이용한 출력(output) 방법을 몇가지 볼게요. 아래는 대표적 출력 방법입니다.

  • Element 요소의 컨텐츠에 출력시 : innerHTML(속성)=value 이용
  • 문서영역(body) 내부에 출력시 : document.write(value);
  • alert(value) : alert() 메서드를 이용한 출력
  • 브라우저 개발자 콘솔에 출력 : console.log(value);
